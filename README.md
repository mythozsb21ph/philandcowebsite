# Philandco Agency Website

A responsive website for Philandco Agency showcasing portfolio, services, testimonials, and contact form.

## Features

- Engaging homepage with hero section
- Clear navigation with smooth scrolling
- Services section with detailed descriptions
- Portfolio gallery
- Client testimonials
- Contact form with validation
- Mobile-responsive design

## Structure

- `index.html` - Main HTML file
- `css/style.css` - Stylesheet
- `js/script.js` - JavaScript for interactivity
- `images/` - Folder for images (add your own images here)

## Usage

1. Open `index.html` in a web browser.
2. Navigate through the sections using the menu.
3. Fill out the contact form to send a message.

## Customization

- Replace placeholder images in `images/` folder with actual project images and hero background.
- Update content in `index.html` to match your agency's information.
- Modify styles in `css/style.css` for branding.

## Browser Support

Works on modern browsers with responsive design for mobile devices.

## Troubleshooting

- If images don't load, ensure they are in the `images/` folder with correct names.
- For contact form, currently shows an alert; integrate with a backend service for actual email sending.
